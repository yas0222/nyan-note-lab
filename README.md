# nyan-note-lab

このリポジトリは `yas0222/nyan-note` をコピーして開発用に調整するための作業場所です。

## 現在の状態

この実行環境では外部ネットワークアクセスが制限されており、`https://github.com/yas0222/nyan-note` を clone できませんでした（HTTP 403）。
そのため、元リポジトリからアプリ一式をコピーする作業は未完了です。

## 次に必要なこと

- `yas0222/nyan-note` の内容をこの環境で参照可能にする
  - 例: 同梱アーカイブの配置、またはネットワーク許可
- 参照可能になり次第、以下を実施
  - アプリ一式のコピー
  - `/nyan-note-lab/` パスへの置換
  - 開発環境表示の追加
  - Firebase を `nyan-note-dev` 用へ設定
  - localStorage キーの `nyan-note-lab` 分離
