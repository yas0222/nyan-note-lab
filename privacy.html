にゃん・ノート プライバシーポリシー

このページは、にゃん・ノートで取り扱う情報と公開範囲について説明するものです。アプリをご利用の前にご確認ください。

取得する情報

  • 猫プロフィール（名前、年齢、性別、毛色・柄、都道府県、市区町村、公開設定、写真など）
  • 日次記録（ごはん、水分、排泄、体重、メモ）
  • 公開プロフィール用データ（publicCats）
  • 技術的な識別情報（匿名認証IDや端末保存用ID）

利用目的

  • 猫の健康記録を継続して管理するため
  • 公開を許可したプロフィールを「みんな」画面に表示するため
  • バックアップや復元などのデータ保護機能を提供するため
  • 不具合確認や安定運用のため（Firebase接続確認を含む）

保存方法

  • cats / records は本人用データとして保存されます。
  • publicCats は公開プロフィール用として分離して保存されます。
  • 端末のローカル保存とFirebase（Firestore / 匿名認証）を利用します。

公開される情報

  • 「みんな」画面に表示されるのは publicCats の公開プロフィール情報のみです。
  • 公開設定に応じた猫名・都道府県・市区町村・年齢・性別・毛色などが表示されます。

公開されない情報

  • records、メモ、写真データは「みんな」画面に公開しません。
  • 本人用データ（cats / records）は公開プロフィールとは別に扱います。

Firebase / GitHub Pages の利用

  • 本アプリはGitHub Pages上で配信される場合があります。
  • 保存・同期のためにFirebase（Firestore / 匿名認証）を利用します。
  • 地域非公開の場合、publicCats には都道府県・市区町村を保存しません。

データ削除依頼

  • 削除希望がある場合は問い合わせ先までご連絡ください。
  • 削除対象の特定に必要な情報（利用時期・公開名など）をご案内いただく場合があります。

バックアップについて

  • アプリ内のバックアップ機能でJSONを書き出し・読み込みできます。
  • バックアップファイルの保管・管理は利用者ご自身で行ってください。

医療に関する注意

  • にゃん・ノートは獣医療上の診断、治療、緊急時判断を行うものではありません。
  • 体調不良や緊急時は、必ず動物病院や専門家に相談してください。

セキュリティ

  • 公開範囲を分離し、必要最小限の情報のみを公開対象として扱います。
  • 本名、住所、電話番号などの個人情報は入力しないでください。
